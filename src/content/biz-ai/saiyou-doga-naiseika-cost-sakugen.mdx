「採用動画は欲しいが、1本数十万円は予算的に厳しい」「年間で見ると採用予算の1〜2割が動画制作で消えている」——採用責任者・経営層からよく聞こえてくる悩みです。

この記事では、採用動画の制作コストが採用予算を圧迫している企業に向けて、外注と内製のROIを並べて試算し、AI内製化への現実的な移行手順を解説します。コスト削減と採用効果の両立をどう設計するかを、稟議に通せるレベルで具体化します。

## 採用動画にかかる「本当のコスト」を分解する

まず、採用動画1本にいくらかかっているのかを正確に把握することから始めます。多くの企業が見積書の「制作費」だけを見ていますが、実際は以下の隠れコストが上乗せされています。

| 項目 | 中小企業の典型額 |
|---|---|
| 制作会社への支払い（企画・撮影・編集） | 30〜80万円／本 |
| 経営者・社員の出演拘束時間 | 半日〜1日／本 |
| 社内調整工数（広報・人事の打ち合わせ） | 5〜10時間／本 |
| 修正・差し替え対応 | 5〜15万円／回 |
| 媒体別の尺調整・カット編集 | 3〜10万円／媒体 |
| 撮影日程の調整・延期コスト | 機会損失として発生 |

例えば「制作費50万円の採用動画」と聞いた時、実際は経営者の半日×5,000円換算の機会損失、人事担当者の10時間分の人件費、修正対応1回分などを足すと、トータル80〜100万円が消えている計算になります。

### 採用ROIで見ると、動画コストはどう影響するか

採用1名あたりのコスト（CPH：Cost Per Hire）の中で、動画制作費が占める割合を見てみます。

中小企業の典型的なCPH内訳：

- 求人媒体費：30〜50万円
- エージェント手数料（年収の30〜35%）：100〜200万円
- 採用担当者の人件費：30〜50万円
- 採用動画制作費：年間200〜500万円 ÷ 採用人数

採用人数が年5〜10名規模だと、動画コストだけで採用1名あたり20〜100万円が乗ります。これは **エージェント手数料に次ぐ重さ** で、CPHの中でもインパクトが大きい項目です。

## 採用ROIを下げる典型パターン

採用動画を作っているのにROIが上がらない企業には、共通するパターンがあります。

### 1. 「立派な1本」に予算を集中させて、媒体別最適化ができていない

100万円かけて作った渾身の1本を、採用サイト・Wantedly・YouTube・TikTokに同じ尺で流している。媒体ごとに視聴者の特性は違うのに、1本で兼用するために、結局どの媒体でも刺さらない動画になっています。

### 2. 動画を更新できず、情報が古くなる

組織体制・募集職種・事業内容が変わっても、動画を撮り直す予算がなく古い動画を使い続けている。求職者にとっては「いつの情報？」と不信感を持たれる原因になります。

### 3. 採用ターゲット別の出し分けができていない

エンジニア向け・営業職向け・新卒向けで、本来は別動画が望ましい。しかし制作コストの都合で1本に集約され、結果として「誰にも刺さらない動画」になっています。

これら3つはすべて **「制作コストが高すぎて柔軟に動かせない」** ことが原因です。コスト構造を変えれば、これらの問題はすべて解決します。

## AI内製化で削減できる費用の構造

AIアバター動画生成を採用動画に活用すると、コスト構造が以下のように変わります。

### 消滅するコスト

- **撮影費（カメラマン・機材・スタジオ）**：撮影自体が不要になる
- **経営者・社員の出演拘束**：スタジオ拘束時間がゼロ
- **撮影日程の調整・延期**：日程調整プロセスそのものが消滅
- **編集の大半**：AI生成時点で完成度の高いMP4が出力される

### 残るコスト

- **AIアバター動画SaaS利用料**：月額3〜10万円程度
- **初回ボイスクローン構築**：10〜30万円（一度きり）
- **原稿作成・運用工数**：人事担当者1〜2時間／本

### 新たに発生するコスト

- **本人合意・ガイドライン整備**：法務との連携工数（初期のみ）
- **AIアバター開示の運用設計**：広報部との連携

差し引きで、年間20本制作する企業の場合、**1,000〜1,500万円規模の削減** が現実的に見込めます。

## ROI試算：外注 vs AI内製（採用5名／年・動画20本）

具体的な数字で比較します。年間採用5名・動画20本制作する中堅企業を想定。

### 外注パターン

| 項目 | 金額 |
|---|---|
| 動画制作費（50万円×20本） | 1,000万円 |
| 経営者・社員の出演拘束（隠れコスト） | 100〜200万円 |
| 修正・差し替え（年5回想定） | 50万円 |
| 採用動画関連の合計 | **約1,150〜1,250万円／年** |
| 採用1名あたり動画コスト | 230〜250万円 |

### AI内製パターン

| 項目 | 金額 |
|---|---|
| AIアバター動画SaaS（月10万円×12ヶ月） | 120万円 |
| 初回ボイスクローン構築 | 20万円 |
| 人事担当者の運用工数（2時間×20本×時給5,000円） | 20万円 |
| 採用動画関連の合計 | **約160万円／年** |
| 採用1名あたり動画コスト | 32万円 |

差し引き **約1,000万円／年の削減**。採用1名あたりに換算すると、200万円のコスト削減になります。これはエージェント手数料1名分に匹敵するインパクトです。

## 内製化フェーズの段階的な進め方

ただし、外注からいきなり全面内製化に移行すると失敗するケースがあります。以下のフェーズ設計を推奨します。

### Phase 1：パイロット（1〜2ヶ月）

社内向けの簡易動画から始めます。例えば「内定者向けの社長メッセージ動画」「内定者懇親会の事前案内動画」など、外部に出ない動画でAI内製の運用感を掴みます。

### Phase 2：スカウトメール添付動画（3〜4ヶ月）

スカウトメールに添付する短尺動画から、対外発信に展開します。視聴者は候補者個人に限定されるため、ブランドリスクが低い領域です。スカウトメールに動画を添付すると返信率が改善するという副次効果も得られます。

### Phase 3：採用サイト・媒体別動画（5〜6ヶ月）

採用サイト・Wantedly・Indeedなど、媒体別に動画を出し分ける運用に展開します。媒体ごとの視聴データを取りながら、原稿の最適化を進めます。

### Phase 4：採用イベント・説明会動画（7ヶ月以降）

最後に、合同説明会・キャリアイベントでの上映動画など、対外的な「公式」動画も内製化します。ここまで来ると、年間の動画制作費は外注時代の1/10以下に圧縮できているはずです。

## コスト削減と採用効果の両立

「コストを下げると採用効果も下がるのでは」という懸念は当然です。しかし実態は逆で、以下の理由から **採用効果は上がる** ケースが多くあります。

### 1. ターゲット別の動画を量産できる

エンジニア向け・営業向け・新卒向けで、それぞれ別動画を出せるようになります。1本の汎用動画より、ターゲットに刺さる別動画のほうが応募率は明確に上がります。

### 2. 情報の鮮度を保てる

組織変更・新規事業立ち上げ・新オフィス開設などのタイミングで、動画を即座に更新できます。「最新の会社」を伝えられることが採用の差別化要因になります。

### 3. 経営者メッセージを毎月発信できる

月次の経営者メッセージ動画を採用候補者に向けて発信することで、入社前から経営者の人柄・思想を伝えられます。これは内定承諾率の改善に直結します。

[社長スタジオ](/cases/nikkei-mj-shacho-studio-keisai)のような経営者特化のAIアバター動画SaaSは、まさにこの「経営者発信の量産」を支える設計になっています。

## まとめ：採用動画は「投資判断の対象」から「日常運用」へ

採用動画は、これまで「年に1〜2本、数百万円かけて作る大きな投資」でした。AI内製化により、これが「日常的に量産する運用業務」に変わります。

コスト削減効果は年間1,000万円規模、採用1名あたりのCPH換算でも数百万円のインパクトがあります。それでいて、採用効果はターゲット別の出し分けにより向上する——コスト削減とROI改善が同時に起きる、稀有な投資判断と言えます。
