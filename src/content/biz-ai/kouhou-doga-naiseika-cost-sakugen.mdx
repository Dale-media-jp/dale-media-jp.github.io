「広報の動画制作費が年々膨らんでいる」「会社紹介動画を1本作るのに半年かけて100万円。費用対効果が読めない」——上場企業・中堅企業の広報責任者から繰り返し聞こえてくる声です。

この記事では、広報の動画制作コストが膨らむ構造的な要因を分解した上で、AIアバター動画生成を活用した内製化で **実際にどれだけ経費削減できるか** を試算します。会社紹介動画・経営者メッセージ動画・採用補助動画など、広報部が抱える動画案件を外注から内製に移行する実務フローを、稟議資料に転用できるレベルまで具体化しました。

## なぜ広報の動画制作コストは膨らむのか

広報部が抱える動画案件は、ここ数年で確実に増えています。会社紹介動画、株主向けメッセージ、IRサマリー、採用補助、メディア対応用のショート動画、SNS用素材——一昔前なら年に1〜2本だったものが、今は月単位で更新が求められるようになりました。

しかし広報部の人員と予算は比例して増えていません。結果として、1本あたりの単価×案件数で予算が爆発するという構造になっています。

### 外注見積書の内訳を分解する

制作会社からの見積書を分解すると、典型的に以下の構成になっています。

| 項目 | 一般的な相場（5分動画） |
|---|---|
| 企画・構成費 | 10〜20万円 |
| 撮影費（カメラマン・機材・スタジオ） | 15〜30万円 |
| 出演者拘束費（経営者・社員） | 社内コストで見えない |
| 編集費 | 15〜25万円 |
| ナレーション・BGM・テロップ | 5〜10万円 |
| 修正対応（2〜3回） | 5〜15万円 |
| **合計** | **50〜100万円／本** |

ここで見落とされやすいのは「出演者拘束費」です。経営者がスタジオに半日拘束された時間の機会損失を時給換算すると、それだけで数十万円の隠れコストが発生しています。

### 修正・差し替えの非対称コスト

もう1つの構造的問題は、修正・差し替えコストの高さです。「メッセージのトーンを変えたい」「事業内容が変わったので録り直したい」と思っても、再撮影には数十万円の追加コストと数週間の調整期間が必要になります。

結果として、古くなった動画を使い続けるか、思い切って撮り直すかの極端な二択になり、広報の機動力が削がれます。

## 動画内製化の3パターンと費用感

「内製化」と一言で言っても、実態は3つのパターンに分かれます。

**1. フル内製（撮影〜編集まで自社）**
社内に動画制作チームを設置するパターン。年間で何十本も制作する大企業向けで、機材投資・人件費・スキル教育を含めると初年度で2,000〜3,000万円規模の投資になります。

**2. ハイブリッド内製（撮影は外注、編集は内製）**
撮影はカメラマンに外注し、編集は広報部内で実施するパターン。1本あたり20〜40万円に圧縮できますが、撮影日程の調整と経営者拘束問題は残ります。

**3. AI内製（撮影レス・テキスト入力で生成）**
[社長スタジオ](/cases/nikkei-mj-shacho-studio-keisai)のようなAIアバター動画生成SaaSを使い、原稿テキストから本人の顔・声で動画を生成するパターン。撮影フェーズが消滅するため、コスト構造が根本から変わります。

3パターンの中で、年に5〜30本の動画を扱う中堅・上場企業の広報部に最も適合しやすいのが、3番目のAI内製です。

## AI内製化で実際にどれだけ削減できるか

ここからが本題です。AIアバター動画生成を導入した場合、広報の動画制作コストはどう変わるのか。年間20本（5分動画）を制作する広報部を想定して試算します。

### 外注の場合（年間20本）

| 項目 | 単価 | 年間 |
|---|---|---|
| 制作費（外注） | 70万円／本 | 1,400万円 |
| 経営者拘束（隠れコスト） | 半日×20回 | 100〜200万円 |
| 修正・差し替え | 平均1.5回／本 | 150万円 |
| **合計** | | **約1,650〜1,750万円／年** |

### AI内製の場合（年間20本）

| 項目 | 単価 | 年間 |
|---|---|---|
| AIアバター動画SaaS | 月額制 | 30〜60万円 |
| 初回ボイスクローン構築 | 一括 | 10〜30万円 |
| 広報担当者の運用工数 | 1本あたり2時間×時給5,000円 | 20万円 |
| 修正対応（テキスト差し替えのみ） | 工数のみ | 10万円 |
| **合計** | | **約70〜120万円／年** |

差し引きで **1,400〜1,650万円／年の削減**。広報担当者1名分の人件費を上回るインパクトになります。

### コスト削減以上のメリット

数字に出ない部分でも変化があります。

- **修正の心理的ハードル消滅**：原稿テキストを書き換えるだけで再生成できるため、「メッセージを最新化する」が日常業務になる
- **ターゲット別の動画展開が可能**：採用ターゲット別、メディア別、地域別など、本来は分けたい動画を低コストで作り分けられる
- **多言語展開が現実的になる**：本人の声質を保ったまま英語・中国語などへの吹き替えができ、海外向け広報の幅が広がる

## 広報部のAI内製化フロー

実際に外注から内製に移行する場合の標準フローです。

### Phase 1：パイロット導入（1〜2ヶ月目）

最初から全動画を切り替えるのではなく、社内向け動画（朝礼用社長メッセージなど）から始めるのが定石です。失敗してもダメージが少なく、運用ノウハウを蓄積できます。

1. AIアバター動画SaaSの選定（HeyGen / Synthesia / 社長スタジオなど）
2. 経営者の写真・音声サンプルの収録（初回30分程度）
3. ボイスクローン・アバター学習（数日）
4. テスト動画3〜5本生成・社内レビュー

### Phase 2：社内向け動画への展開（3〜4ヶ月目）

朝礼動画、月次方針共有、社内表彰動画など、視聴者が社員に限定される領域に展開します。失敗してもブランド毀損リスクが低く、実運用の精度を上げる期間として機能します。

### Phase 3：対外発信動画への展開（5〜6ヶ月目）

会社紹介動画、採用補助動画、SNS用ショート動画など、対外的なメッセージに広げていきます。この段階では、外部レビュー（経営層・法務）のフローを必ず挟むようにしましょう。

### Phase 4：IR・株主向け動画（7ヶ月目以降）

最後に、IR動画・決算説明会のサマリー動画など、最も慎重さが求められる領域に展開します。ここまで来ると、動画制作の主導権は完全に広報部に戻り、外注は「特別な1本」だけに限定できる状態になります。

## 内製化で気をつけるべき落とし穴

コスト削減効果は大きい一方、いくつかの注意点があります。

**1. AIアバターであることの開示**
本人の声・顔をAIで再生成する以上、視聴者に対して「AI生成動画である」ことを明示するか、社内ガイドラインで開示ポリシーを決めておく必要があります。

**2. 経営者本人の最終確認フロー**
原稿の段階で必ず経営者本人がレビューする運用を組み込みましょう。AIで作りやすくなるからこそ、メッセージの「本人らしさ」が運用の生命線になります。

**3. ブランドガイドラインとの整合**
動画の背景画像・テロップ・色使いが既存のコーポレートブランドガイドラインと整合しているかを、初期設定時に必ず確認します。

**4. 本人合意書の整備**
経営者の声・顔をAI学習に使う以上、利用範囲・期間・第三者提供の有無を明文化した合意書が必要です。導入時に法務と協議して整備してください。

## まとめ：広報の動画制作は「経費」から「インフラ」へ

動画制作費を「都度発生する経費」として外注に発注する時代から、「常時動いている社内インフラ」として内製化する時代に移っています。AIアバター動画生成は、この移行を経済合理性の面から後押しする技術です。

広報の動画予算を **年1,000万円単位で削減しながら、制作本数は逆に増やす**——この両立を可能にするのが、AI内製化の本質です。
